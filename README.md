# Калькулятор тарифов CTT.BY (EDI-провайдер СТТ)

Статическое веб-приложение для быстрого и удобного расчета стоимости услуг электронного документооборота (EDI) от провайдера «Современные технологии торговли» (СТТ).

## Описание проекта

Этот проект представляет собой интерактивный калькулятор тарифов, предназначенный для подсчета стоимости использования электронных накладных (ЭТН, ЭТТН), маркировки и других EDI-сообщений. Приложение работает на клиентской стороне и не требует серверной части для вычислений, что обеспечивает моментальный отклик интерфейса.

## 🚀 Развертывание

Проект настроен на автоматический деплой с помощью **GitHub Pages** через пайплайн GitHub Actions (`static.yml`). 

**Актуальная версия калькулятора доступна по ссылке:**  
👉 [https://nikulenka.github.io/CTT-Tarifs/](https://nikulenka.github.io/CTT-Tarifs/)

## 🛠 Стек технологий

- HTML5 / CSS3
- Vanilla JavaScript
- GitHub Actions (CI/CD)
- GitHub Pages (Хостинг статического контента)

## 🔄 Поддержка и обновление

Любые коммиты и изменения, внесенные в основную ветку `main` (например, при обновлении прайс-листа `Price.pdf` или изменении логики расчета), автоматически запускают workflow в GitHub Actions. Изменения компилируются и публикуются на сайте GitHub Pages в течение нескольких минут без необходимости ручного вмешательства.
